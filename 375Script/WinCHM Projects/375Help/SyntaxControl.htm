Loop

Loops the script from the first line.

Stop

Stops the script.

Stop:all

Stops this script and all looping scripts. Waits 5 seconds for them to stop.

Stop:others

Stops other looping scripts. Waits 5 seconds for them to stop.

Wait

Pauses the script for the specified seconds. Allows up to 3 decimal places.

Waituntil

Pauses the script until the specified time.
