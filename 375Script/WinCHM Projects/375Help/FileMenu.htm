New

Clears the editor. Prompts you to save if unsaved.

Open

Prompts you to open a file. Prompts you to save if unsaved.

Open Link

Prompts you to open a file. Prompts you to save if unsaved.

Will open link files (.lnk, aka shortcuts), instead of deferencing them.

Reload

Reloads the file, will not do anything if no file is opened.

Save

Saves the file, does the same as Save As if not saved before.

Save As

Prompts you to save a file.

Page Setup

Currently does not do anything.

Print

Prints the file. However this is not tested before.

Exit

Closes the program. Stops executing scripts.
