Status Bar

Toggles the status bar's visibility.

Debug Mode

Does the same as Open RTFDebug function.

Syntax Mode

Does Open cRTFSyntax function properly.

Zoom In

Zooms the script by twice.

Zoom Out

Zooms the script by half.

Reset Zoom Level

Resets the zoom level to 1.

Set Zoom

Sets the zoom level to the input.

Note: 

The zoom level value be between 1/64 (0.015625) and 64.0, not inclusive. A value of 1.0 indicates that no zoom is applied.

Therefore, the zoom level limits are actually 0.0156250019 and 63.9999962 due to the error (the term in Mathematics) of single-precision floating point numbers (professional words, eh?).
